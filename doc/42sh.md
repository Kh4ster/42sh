42sh(1) -- Interactive Shell coded in C
=======================================

## SYNOPSIS

`42sh` [<optional>...] <flags>

`42sh` [<optional>...] <flags>

`42sh` [<optional>...] <flags>

`42sh` [<optional>...] <flags>

## DESCRIPTION

**42sh** is an interactive shell reproducing basic functionalities of bash.
